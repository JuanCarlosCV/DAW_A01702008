URL

Es una direccion utilizado para localizar recursos en internet. El destino puede ser una pagina de web, archivo de texto o un programa.

MIME

Extensiones multiproposito de Correo de internet. Esta definido por indicar la naturaleza y el formato del documento

Metodos y codigos de errores HTTP

Son utilizados para saber si se ha cumplido una solicitud especifica de HTTP, ejemplos tipos: respuestas informativas, respuestas satisfactorias, redirecciones, errores de clientes y servidores

XML

Este tipo de lenguaje esta destinado a paginas que se publican en la web , es una forma simplicada de SGML(lenguaje de marcado generalizado generalizado).

XHTML

Es un lenguaje similar a HTML, en simples palabras es una adaptacion de HTML al lenguaje XML.

JS

Es un lenguaje de programacion que es utilizado frecuentemente para agregar funciones interectativas en los sitios web. Entre algunas de las funciones puede ser juegos, efectos dinamicos y animacion.

PHP

En el Mundo de los programadores en un lenguaje muy comun y de codigo abierto, esta enfocado en desarrollo web y puede ser anexado a HTML facilmente.

JSP

Es una herramienta utilizada en el mundo del desarrollo web, con el fin de facilitar la creacion de contenido web dinámico. Tiene la caracteristica de trabajar independientemente del servidor y de la plataforma.

AJAX

Se describe por ser una herramienta que trabaja en conjunto con HTML o XHTML, CSS,JAVASCRIPT,DOM,XML,XSLT,y el objeto XMLHtttpRequest. Es importarte resaltar que no es una tecnologia independiente, con esta misma es posible lograr aplicaciones web capaces de actualizarce continuamente sin tener que volver a cargar la pagina.

¿Cuál es la diferencia entre Internet y la World Wide Web?

Se definen por una gran diferencia, internet es toda la union de las computadoras existentes, en pocas palabras todo lo que conocemos como internet. WWW es definido por ser solo una parte de internet, es una parte muy importante, pero esta dentro del mundo de internet.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Es utilizado para leer un resource, y devulve el recurso en ese destino.

HEAD: Es utilizado para comprobar el sitio destino, por ese motivo solo se recibe un codigo de respuesta.

POST: Es utilizado con la implementacion de formularios

PUT:Es utilizado para actualizar contenidos

PATH: Es utilizado como el put, para modificacion de un recurso.

DELETE: Es usado para eliminar un resource

Este error es debido a un falta de un recurso colocado incorrectamente o movido, debe ser corregido por el programador
