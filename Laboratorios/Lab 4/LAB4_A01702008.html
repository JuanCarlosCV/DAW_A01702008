¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Tiene algunos cambios en tipos de datos, pero en su sintaxis. En la mayoria de las operaciones trabajan de igual manera. Tambien alguna de las diferencias es el enfoque en el que se trabaja, javascript es para paginas web y java es para codigo en aplicaciones escritorio

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

Date.now() : devuelve el valor de la hora actual

Date.parse(): transforma una cadena de una fecha y devuelve el numero el milisegundos

Date.UTC(): Acepta los mismos parámetros que la forma más larga del constructor, y devuelve el número de milisegundos en un objeto Date desde el 1 de enero de 1970, hora universal 00:00:00.

Date.getFullYear(): devuelve el año

Date.getMonth(): Devuelve el mes
