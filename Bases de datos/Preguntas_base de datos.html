¿Qué es entonces una base de datos - DB?

Una base de datos es una colleccion en la cual se almacena diferentes tipos de datos, entre algunos de ellos podrian ser, Ejemplo: Alumnos, maestros, libros,etc Este tipo datos se almacen conforme a sus atributos principales y se mantiene un orden de acuerdo a como se haya diseñado,su principal objetivo es mantener una relacion entre la misma coleccion, permitiendo asi un mejor control y administracion de ella.

¿En qué casos es conveniente usar bases de datos?

Es conveniente utilizar bases de datos en empresas de manufactura o alguna organizacion. Con informacion como ventas, compras, produccion, finanzas, etc.

¿Qué es un sistema de gestión de base de datos - DBMS?

Este concepto lo defino por el uso de base de datos, pero en conjuntos con programas para su manipuacion. Esto quiere decir que se toman en cuenta el uso de toda la informacion, accesos de cuentas, la seguridad de la informacion y una buena manipulacion de los datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Creacion de una base de datos: Es la modelacion de la collecion de datos

Establecimiento de una comunicacion entre interfaz-base de datos: Es el modo de conexion entre el administrador de datos y la pagina

Manipulacion y registro de datos: Debe existir formas de registrar, cambiar o eliminar datos

Controles de concurrencia: Es el manejo de informacion actualizada entre cambios

Seguridad e integridad: Manejar niveles de seguridad para cambios inoportunos
